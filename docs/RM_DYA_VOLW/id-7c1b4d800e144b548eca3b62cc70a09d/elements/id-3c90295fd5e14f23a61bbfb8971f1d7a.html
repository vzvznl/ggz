Geïntegreerd gebruik van architectuurtools ()
De architecten worden ondersteund door een geïntegreerde set van tools waarmee onder andere de onderlinge consistentie van architecturen kan worden gecontroleerd. Controlepunten - De gebruikte architectuurtools zijn met elkaar geïntegreerd. - Met behulp van de architectuurtools kan de onderlinge consistentie van architecturen worden gecontroleerd. Verbetersuggesties - Integrale toolkit invoeren. Implementeer een integrale toolkit, waarmee niet alleen de verschillende architecturen kunnen worden vastgelegd, maar, door integratie van de tools, ook onderlinge consistentie kan worden bewaakt. Een alternatief is natuurlijk één tool dat alle aspecten afdekt.
