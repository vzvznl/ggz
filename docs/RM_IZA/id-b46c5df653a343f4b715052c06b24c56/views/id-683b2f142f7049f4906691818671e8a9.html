In dit model probeer ik het Integraal Zorg Akkoord te ontrafelen en inzichtelijker te maken door deze weer te geven in visuele samenhang en relaties. - Dit inzicht kan onze VZVZ sectorteams helpen bij het kiezen van de juiste focus op thema's die de sectoren het meest raken. - Dit inzicht kan de sectoren zelf wellicht zelf ook helpen - bijvoorbeeld bij het maken van hun jaarplannen.
