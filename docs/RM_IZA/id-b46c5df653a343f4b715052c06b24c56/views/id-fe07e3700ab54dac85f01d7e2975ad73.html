Twiin ()
Twiin ontwikkelt een landelijk afsprakenstelsel om gegevens te delen tussen zorgaanbieders over meerdere infrastructuren heen. Daarbij wordt gebruik gemaakt van een aantal gemeenschappelijke voorzieningen. Website: https://www.twiin.nl/ Architectuur: https://public.vzvz.nl/ Huidige versie: 1.1 beta
