Digitale architectuur voor sectorteam GGZ

Binnen deze repository host ik een architectuurmodel die mij helpen bij de ondersteuning van het sectorteam GGZ van de Vereniging van Zorgaanbieders Voor Zorgcommunicatie (VZVZ). Het sectorteam GGZ is gericht op de ondersteuning van de sector GGZ waar het gaat om hulp bij het realiseren van interoperabiliteit. Als vertrekpunt hanteer ik daarbij steeds Referentiedomeinmodel voor de GGZ (RDG).
