02. Samenhang tussen bedrijfsactiviteit, bedrijfsproces en bedrijfsfunctie ()
In dit model heb ik het plaatje uit de pdf getracht te vertalen naar een geldige Archimate relatie. Ik doe de aanname dat een bedrijfsfunctie de activiteiten bevat die hergebruikt worden door de processen. - Een bedrijfsactiviteit is slechts aan één bedrijfsfunctie gekoppeld. - Een bedrijfsactiviteit kan hergebruikt worden binnen meerdere processen. - Informatie wordt verwerkt binnen bedrijfsactiviteiten.
