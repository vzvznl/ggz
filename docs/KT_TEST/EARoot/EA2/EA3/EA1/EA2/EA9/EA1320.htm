De gebruiker gebruikt een Client Applicatie, dit kan elke willekeurige applicatie zijn waartoe de gebruiker heeft besloten. Aangenomen wordt dat de gebruiker deze cliënt vertrouwt voor interactie met de onderliggende dienst(en). De Authorization Server heeft op zijn minst weinig vertrouwen in de Client Applicatie wanneer de cliënt openbaar of semi-vertrouwelijk is. Aangenomen wordt dat de cliënt op de hoogte is van de specificaties van de FHIR RESTful API en dat autorisatie vereist is. De cliënt gebruikt meestal een Browser, als grafische user interface, of relevante onderdelen die  als user interface geïntegreerd zijn in de gebruikerstoepassing.<br/><br/>Een Client Applicatie MOET de autorisatie flow van OAuth2 (<b>RFC6749</b>) gebruiken om toegang te krijgen tot de gegevensstroom.<br/>
