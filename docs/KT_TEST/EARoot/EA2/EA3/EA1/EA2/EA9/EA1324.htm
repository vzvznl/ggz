De FHIR (Resource) Provider wordt geleverd door een (dienstverlenende ICT) organisatie. Aangenomen wordt dat de FHIR (Resource) Provider bekend is (bij de Autorisatie) voordat daadwerkelijke autorisatie van de gebruiker en/of systemen plaatsvindt.  <br/>Aangenomen wordt dat een FHIR (Resource) Provider wordt geleverd via HTTPS met behulp van TLS, andere protocollen vallen buiten het bereik van dit profiel.<br/>
