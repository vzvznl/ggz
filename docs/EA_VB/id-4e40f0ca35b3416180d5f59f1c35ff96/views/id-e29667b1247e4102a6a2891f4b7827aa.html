02. Hoe Archimate toegepast wordt ()
Bij het toepassen van Archimate probeer ik altijd binnen de lijntjes van de regels van de taal te blijven. Als basis probeer ik altijd gebruikt te maken van volledige patronen zoals ze hierboven zijn weergegeven. In veel gevallen echter kan of wil ik deze patronen niet volledig invullen. In dat geval maak ik gebruik van wat in Archimate "Derivation Rules" genoemd wordt. Van grotere modellen maak ik dan kleinere. https://pubs.opengroup.org/architecture/archimate3-doc/ch-relationships-Normative.html#sec-Specification-of-Derivation-Rules
