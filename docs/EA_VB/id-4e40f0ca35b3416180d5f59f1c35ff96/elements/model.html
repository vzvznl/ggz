EA - voorbeeld
Deze publicatie bevat een voorbeeld van een enterprise archtectuur waarbij dienstverlening als uitgangspunt is genomen. De scope van de enterprise omvat zo veel mogelijk op zorg gerichte diensten - landelijk, en internationaal. Het doel van de architectuur is gezien de brede scope vooral informatief en niet richtinggevend. Gewenste uitkomsten van deze architectuur zijn: - Het voor mijzelf iets beter de bomen door het bos zien - Voor collega architecten wellicht hetzelfde - Complexe beschreven onderwerpen vertalen naar een veel minder complex uniform dienstverleningsmodel
